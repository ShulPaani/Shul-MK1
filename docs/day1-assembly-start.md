# 🤖 6 DOF Robot Arm Project

## 📅 Day 1: Servo Calibration (Centering to 90°)

### 🎯 Objective
Initialize and calibrate all servos to their neutral (90°) position before starting mechanical assembly.

---

### 🔧 Work Done
- Used **3× MG996R** and **3× MG90S** servos
- Connected all servos to the controller ( Arduino Nano ) one by one 
- Ran a basic control script to set each servo to **90 degrees**
- Mounted servo horns after confirming center position

---

### 🧠 Why This Step is Important
- Ensures **proper alignment** during assembly  
- Provides **equal range of motion** on both sides  
- Prevents **mechanical stress and misplacement**  
- Makes future calibration and control easier
